= 推奨されるFSx for ONTAPファイルシステムを導入
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
推奨される FSx for ONTAPファイルシステム (場合によっては複数のファイルシステム) が要件を正確に満たしていることを確認したら、Workload Factory を使用して AWS 環境にシステムをデプロイできます。

以下の指示に従ってくださいlink:https://docs.netapp.com/us-en/workload-fsx-ontap/create-file-system.html["NetApp Workload Factory で FSx for ONTAPファイルシステムを作成する"^]Workload Factory が推奨する構成に従って FSx for ONTAPファイルシステムを展開します。手順の*高度な作成*手順を使用して、ファイル システムを完全にカスタマイズできます。
